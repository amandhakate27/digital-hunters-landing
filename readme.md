## Features

- Hero section with dynamic background and image positioning
- Glass-morphism design for stats section
- Horizontal scrolling project carousel
- Process workflow visualization
- CTA (Call-to-Action) section with overlay effect
- Footer with social media integration
- Smooth scrolling and transitions
- Remix Icon integration for iconography
